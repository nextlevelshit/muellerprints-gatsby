  <h2>Druckdaten für den Digitaldruck</h2>
  <br/>
  <h3>Benennung der Druckdaten</h3>
  <ul>
    <li>Es ist wichtig, dass Sie Ihre Druckdateien eindeutig benennen.
      Also nicht „Flyer“ oder „Bachelorarbeit“ oder „Buch“, sondern
      „M_Müller_flyer_final_print_2018“.
    </li>
  </ul>
  <h3>Dateiformate</h3>
  <ul>
    <li>Liefern Sie uns optimalerweise ein druckfertiges, druckfähiges PDF
      - <b>PDF/X-3:2003</b> oder <b>PDF/X-4:2010</b></li>
    <li>Oder offene Programmdateien von Microsoft Office, <b>Adobe InDesign CC</b>, <b>Adobe Illustrator CC</b>, <b>Adobe
      Photoshop CC</b>.
    </li>
    <li> Wenn Sie uns die offenen Programmdateien übergeben, dann wandeln Sie entweder alle Schirften in Pfade um,
      bzw.
      stellen Sie uns alle verwendeten Schriften zur Verfügung und betten Sie alle Bilder ein, bzw. schicken Sie
      alle
      verknüpften Bilder mit.
    </li>
  </ul>
  <h3>Beschnittzugabe</h3>
  <ul>
    <li>Falls Flächen oder Bilder ganzformatig, randlos dargestellt werden sollen, sollten Sie eine Beschnittzugabe
      von
      3mm an allen vier Seiten anlegen. Flächen oder Bilder müssen also 3mm über den Rand (das Format) hinausragen.
      Soll das beschittene Endformat 210x297mm sein, ist das zu druckende Rohformat 216x303mm.
    </li>
  </ul>
  <h3>Schnittmarken</h3>
  <ul>
    <li>Generell benötigen wir keine Schnittmarken. Falls Sie trotzdem Marken einfügen, sollte der Abstand der
      Marken
      zum Endformat (<b>Versatz</b>) genau so groß sein, wie die Beschnittzugabe. D.h. bei einer Beschnittzugabe
      von 3mm
      muß der Versatz ebenfalls 3mm betragen, bei 18mm Beschnitt ist der Versatz auf 18mm einzustellen.
    </li>
    <li>Und - bitte nur Schnittmarken auswählen!</li>
  </ul>
  <h3>Farbprofile</h3>
  <ul>
    <li>Verwenden Sie eines der folgenden Profile:</li>
    <b>ISOcoated_v2_eci.icc / CoatedFOGRA27.icc / CoatedFOGRA39.icc</b>

  </ul>
  <h3>Farbraum</h3>
  <ul>
    <li>Das beste Druckergebnis erhalten Sie, wenn Sie alle <b>Bilder im Photoshop in CMYK </b>konvertieren.
    </li>
  </ul>
  <br/>
  <h3>PDF Exportvorgabe <small>(Kurzfassung)</small></h3>
  <b>Standard:</b> PDF/X-4:2010
  <br/>
  <h3>Allgemein</h3>
  <ul>
    <b>Bereich:</b> Alle Seiten
    <br/><b>Seiten</b> (nicht Druckbögen)
    <br/><b>Komprimierung</b> &rArr; wie Voreinstellung
    <br/>
    <b>Marken und Anschnitt</b> &rArr; <b>Schnittmarken</b> &rArr;
    <b>Versatz</b> = 3mm &rArr; <b>Anschnitt</b> oben, unten, links, rechts = 3mm
    <br/><b>Ausgabe</b> &rArr; <b>Farbe</b> &rArr; <b>In Zielprofil konvertieren</b> (Werte
    beibehalten)
    <br/><b>Ziel: ISOcoated_v2_eci.icc</b>
    <br/><b>PDF/X</b> &rArr; Name des Ausgabeprofils = <b>ISOcoated_v2_eci.icc</b>
    <br/>[Wenn Sie <b>ISOcoated_v2_eci.icc</b> nicht zur Verfügung haben, ersetzen Sie es
    einfach
    durch <b> CoatedFOGRA27.icc </b> oder
    <b> CoatedFOGRA39.icc</b>]
    <br/><b>Exportieren</b>
  </ul>
